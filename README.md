### 简述下你理解的优雅降级和渐进增强
* 优雅降级：先发布最新浏览器的可运行版本，再向低版本兼容
* 渐进增强：以内容为主，先向低版本浏览器做兼容性处理，再完善新版本浏览器内容支持
